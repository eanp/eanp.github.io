# AyatPocket

A lightweight, mobile-friendly Quran viewer that allows users to read the Quran with smooth page navigation.

## Features

- 📱 Responsive design that works on both mobile and desktop
- 🔖 Automatic page position saving
- ⌨️ Keyboard navigation support
- 🚀 Lazy loading of images for better performance
- 📍 Remembers last read page using localStorage
- 🖼️ High-quality Quran pages from quran.ksu.edu.sa

## Technical Details

- Built with vanilla JavaScript
- Uses tiny-slider for smooth page transitions
- Implements lazy loading for optimal performance
- No backend required - static file serving only
